• PHP Q1

    To display 40 out of 200 characters you would do:
    $string = "This is an example and not 200 characters long"
    echo substr($string,0,39);
    In this case you would replace 0 with the character you want to start from and 39 the character you want to end on.

  • PHP Q2

    To remove all numbers from the string you would do:
    $string = "This 89 is a54n ex9mpl3e";
    echo preg_replace("/[0-9]/", "", $string);
    This replaces all numbers 0 to 9 with white space.

  • PHP Q3

    To replace the characters so the message is decrypted you can do:
    $message = "a r d";
    $CharsOne = array("a","r","d");
    $CharsTwo = array("c","m","e");
    $string = str_ireplace($CharsOne,$CharsTwo,$message);
    echo $message;

    This puts the new letters inplace of the old letters.

  • PHP Q4

    To get the domain of the url you would do:
    $url = "example of url here";
    var_dump(parse_url($url, PHP_URL_HOST));

  • Q5 PHP

    To get the name and value of the parameters you should:
    $url = "example of url here";
    $url_parts = parse_url($url);
    parse_str($url_parts['query'], $params);
    echo 'Your name is'.$params['name'].'. Your age is'.$params['age'];
